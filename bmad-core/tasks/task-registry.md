# BMAD Task Registry

This file tracks standard tasks available in the BMAD workflow.

## Discovery Tasks
- `gather-vision` - Elicit project vision and goals from stakeholder
- `identify-stakeholders` - Map all stakeholders and their interests
- `define-scope` - Establish in-scope and out-of-scope boundaries
- `create-project-brief` - Produce the project brief document

## Analysis Tasks
- `gather-requirements` - Detailed requirements elicitation
- `write-user-stories` - Create user stories with acceptance criteria
- `create-domain-model` - Model the business domain
- `prioritize-requirements` - Apply MoSCoW prioritization

## Architecture Tasks
- `design-system` - Create system architecture design
- `select-technology` - Evaluate and choose technology stack
- `design-data-model` - Create data model / ERD
- `define-api-contracts` - Design API specifications
- `create-adr` - Document an architecture decision

## Planning Tasks
- `create-sprint-plan` - Plan a sprint with backlog items
- `estimate-tasks` - Estimate effort for tasks
- `map-dependencies` - Identify task dependencies
- `create-timeline` - Build project timeline

## Implementation Tasks
- `implement-feature` - Code a specific feature
- `review-code` - Perform code review
- `refactor` - Refactor existing code
- `fix-bug` - Diagnose and fix a bug

## Testing Tasks
- `create-test-plan` - Write test plan for a feature/sprint
- `write-test-cases` - Design test cases
- `execute-tests` - Run tests and document results
- `report-bug` - Document a defect

## Review Tasks
- `sprint-review` - Conduct sprint review
- `retrospective` - Run sprint retrospective
- `metrics-review` - Analyze quality and velocity metrics

## Deployment Tasks
- `prepare-release` - Prepare release package and notes
- `deploy` - Execute deployment
- `validate-deployment` - Post-deployment validation

## Church Leadership Tasks (Logos Pulse)

### Bishop Tasks
- `diocese-report` - Generate region-wide spiritual health report across all pastors
- `review-pastor` - Deep review of a specific pastor's congregation metrics
- `ordain-pastor` - Create and configure a new pastor account
- `set-diocese-goals` - Define quarterly spiritual KPIs for the diocese
- `compare-congregations` - Benchmark all congregations side by side
- `bishop-escalation` - Handle a crisis case escalated from Sr. Pastor

### Sr. Pastor Tasks
- `church-report` - Full congregation health dashboard
- `plan-sermon-series` - Design a multi-week sermon series with scripture references
- `assign-pastor` - Assign a pastor to a member group or congregation
- `discipleship-plan` - Generate a 30/60/90-day spiritual growth plan
- `staff-review` - Review all pastors under Sr. Pastor oversight
- `set-church-vision` - Define the annual spiritual theme and goals

### Pastor Tasks
- `member-report` - Full spiritual health report for one prayer warrior
- `low-engagement-list` - Identify members with declining prayer/reading stats
- `assign-prayer` - Assign a specific prayer burden to a member
- `send-whatsapp-report` - Generate and send WhatsApp report to a member
- `review-confessions` - Review all pending confession submissions
- `group-summary` - Summary of all members in pastoral care this week
- `pastoral-followup` - Flag and plan follow-up for at-risk members

### Bible Knowledge Tasks
- `passage-lookup` - Look up a Bible passage with context and application
- `find-passages-for-topic` - Find relevant scriptures for a given theme or struggle
- `reading-plan` - Generate a custom Bible reading plan (30/90/365 days)
- `sermon-scripture` - Find key passages for a specific sermon topic
- `doctrine-check` - Verify theological soundness of a teaching against scripture
